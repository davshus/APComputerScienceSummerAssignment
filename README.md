# AP Computer Science Summer Assignments - David Shustin

## Prerequisites

* JRE
* JDK
* AppletViewer (should be installed with the JRE/JDK)
* GNU Make

## Compilation

### All
`$ make all` OR `$ make`

### Projects only
`$ make projects`

### TPS only
`$ make tps`

### Nim only
`$ make nim`

## Run

### Run Nim
`$ appletviewer index.html`

### Run anything else
`$ java <projects or tps>.<classname>`
